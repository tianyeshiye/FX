Open the folder where you installed your MT4 program and follow the procedures below:
1. Create a folder under the directory of Indicators and then place .ex4 files under the folder;
2. Place .tpl file under the directory of Templates;
3. Run MT4 software and right click on a chart, then click load template to load the .tpl file under the directory.

1.运行MT4程序,按键盘F4打开MetaEditor;鼠标右键点击Indicators的文件夹,打开文件夹;在文件夹里新建一个文件夹,把附件中的.ex4文件复制到里边去;
2.把附件中的.tpl文件复制到Templates的文件夹下;
3.回到MT4软件,在要加载模板的图形上点击右键,在弹出的菜单里左键点击Template->Load Template加载复制的.tpl文件.
